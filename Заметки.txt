После слияния веток

git checkout Название основного бранча, в нашем случае develop

git fetch origin

git pull



После того как ветка была соединена с основной веткой ее можно удалить




На удаленном репозитории командой 

git push origin -delete Название ветки



На локальном репозитории

git branch -d Название ветки

или

git branch -D Название ветки


Отличие -d удаление в том случае, если ветка была соединена с основной веткой командой merge


-D удалит даже если ветка еще не была соединена с главной веткой
